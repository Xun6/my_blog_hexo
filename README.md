# my_blog_hexo
## Usage
```shell
# clone this repository
git clone https://github.com/Xun6/my_blog_hexo.git
```

## 安装hexo
> $ npm install hexo

安装以后，可以使用以下两种方式执行 Hexo：

1、npx hexo <command>
2、将 Hexo 所在的目录下的 node_modules 添加到环境变量之中即可直接使用 hexo <command>：
> echo 'PATH="$PATH:./node_modules/.bin"' >> ~/.profile
